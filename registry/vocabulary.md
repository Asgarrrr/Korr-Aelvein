# Vocabulary of Korr Aelvein

Words, names, glyphs, songs. **Gameplay-bearing, not decorative**: knowing a word potentially unlocks an affordance in the game (see `systems.md` § 2 Knowledge / Names / Glyphs).

---

## Proper nouns — decided

### Korr

L'archive. Le nom que l'abysse porte dans les fragments les plus anciens trouvés en descente. Pas traduit. Pas expliqué. Le joueur le trouve ; il comprend d'abord que c'est un nom, puis ce que ce nom désigne.

Usage dans le jeu : "Korr" apparaît dans les glyphes les plus profonds, dans le journal de Mael, dans les formules liturgiques du prêtre dont le sens a été perdu. C'est un mot que les personnages utilisent sans comprendre pleinement ce qu'il signifie.

### Aelvein

Le geste de descendre en sachant qu'on ne revient peut-être pas. Pas un lieu. Un état d'être, un acte volontaire, une relation au risque.

Usage courant dans la ville :
- "Il est parti en Aelvein" = il est descendu et n'est pas revenu (décès présumé)
- "Faire l'Aelvein" = descendre (dans les dialectes plus anciens : descendre rituellement)
- "Quelqu'un d'Aelvein" = quelqu'un qui est revenu de descente (sous-entendu : changé)

Le mot porte à la fois l'acte et sa conséquence possible. Il n'a pas d'équivalent exact en français ou en anglais. Ne pas le traduire.

## Naming mechanics — rules

Nommer une créature dans Korr signifie lui imposer une nature. L'archive enregistre par forme et par moment — pas par nom. Les créatures n'ont pas de noms intrinsèques. Ce sont les descenders qui leur en donnent, et ces noms persistent.

**Un nom complet** est la combinaison de :
1. La **forme** (ce que l'archive a enregistré — un cerf, un homme, un acte)
2. Le **moment** (dans quel état ou acte l'original a été enregistré — la noyade, la dispute, le travail)

Exemple : "Le forgeron perçoit les os — c'est un instrument de mesure." + "Le fossoyeur entend le moment — quelqu'un qui traçait des lignes en silence." = "L'Arpenteur qui-trace-en-silence" ou, dans le vocabulaire du jeu, un nom qui combine les deux informations.

**Le nom comme input** : quand le joueur prononce un nom, c'est un input textuel dans une zone de nommage. Ce qu'il tape doit correspondre à la structure du nom trouvable dans les journaux et observations. Ce n'est pas un code exact — c'est une reconnaissance par correspondance thématique (implémentation à décider).

## Glyphs — système en cours de design

Les glyphes de Korr sont un système d'écriture dont le scribe est le principal lecteur. Ils ne sont pas décoratifs — ils sont fonctionnels dans le sens où ils décrivent l'état de l'archive à l'endroit où ils apparaissent.

Catégories connues :

**Glyphes de statut** — indiquent ce qui se trouve derrière un espace (une zone de haute pression, une zone d'effacement actif, une zone stable). Un scribe qui les lit peut savoir ce qui l'attend sans entrer.

**Glyphes de mémoire** — fragments de ce que Korr a enregistré ici. Le scribe peut les copier. Un fossoyeur peut les "entendre".

**Glyphes de protocole** — les formules fonctionnelles que le prêtre récite sans comprendre. Ce sont des instructions pour Korr, écrites dans un système antérieur à la ville.

## Songs and chants

Les chants du prêtre sont des protocoles. Certains :

- Ralentissent la transition d'une créature de Sondage vers Expression
- Stabilisent un espace (ralentissent l'Endurance ambiante)
- Apaisent une créature en Expression (baissent l'intensité sans changer l'état)

Les chants sont transmissibles uniquement de prêtre à prêtre, oralement. Un prêtre mort sans apprenti emporte ses chants. Les textes liturgiques en contiennent certains mais dans un dialecte que seul un scribe peut lire.

## Names of things in the abyss

Chaque créature du bestiaire a un nom potentiel — pas inné, mais trouvable. La liste de référence se construit à partir du bestiaire (`registry/bestiary.md`) et des journaux des descenders.

Règle de nommage : le nom d'une créature n'est jamais un mot qu'on invente — c'est un mot qu'on trouve. Il existe dans les archives (les journaux, les glyphes, les récits du fossoyeur). Nommer sans trouver, c'est donner un mauvais nom.
